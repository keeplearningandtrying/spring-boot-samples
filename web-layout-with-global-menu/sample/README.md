# Sample Web Application

To demonstrate the usage of the web layout with the global menu.

## Build
```
mvn clean install
```

## Run
```
mvn spring-boot:run
```

Open in the browser:
```
http://localhost:8080/ui/sample/
```

Login:

- User: `demo`/`demo` (can't access the web at all)
- Admin: `demoadmin`/`demo` (full access)
